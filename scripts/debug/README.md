# Scripts de Debugging

Scripts one-time para debugging y verificación de datos.

## Archivos

- `check_*.py` - Scripts de verificación de datos
- `debug_*.py` - Scripts de debugging
- `fix_*.py` - Scripts de corrección one-time
- `recalc_*.py` - Scripts de recálculo
- `test_*.py` - Scripts de testing manual

**Nota:** Estos scripts son herramientas de desarrollo/debugging, no parte del sistema productivo.
